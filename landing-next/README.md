# MeetScheduling Landing (Next.js + Tailwind)

This folder contains a reusable Next.js + Tailwind implementation of the landing page.

## Run

```bash
cd landing-next
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

- `app/page.jsx`: page composition
- `components/*`: reusable section components
- `public/assets/*`: exported visual assets (logo + SVG section visuals)
